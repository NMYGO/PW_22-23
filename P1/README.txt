La ejecución del programa está siempre guiada por una serie de menús que enumeran las opciones de las que dispone el usuario en cada momento. En primer lugar, antes de acceder al propio programa el usuario debe registrarse, o en caso de ya haberse registrado, iniciar sesión. Una vez iniciada sesión, se despliega un menú con las diferentes opciones que ofrece el programa. Cada opción está completamente guiada con los datos que se requieren para esa función específica (en caso de requerirse). Al finalizar el programa, se generarán una serie de ficheros .txt que contendrán información sobre los diferentes datos utilizados en la ejecución del programa, todos los ficheros tienen nombres descriptivos para facilitar la comprensión de su contenido.
**Se ofrecen una serie de ficheros .txt para probar de forma más cómoda el programa, aunque si se quisiese comenzar de cero, se pueden eliminar sin problema**
